Neovim

cheat sheet · normal mode unless noted

Modes

Escback to Normal mode
iinsert before cursor
ainsert after cursor
o/Onew line below / above
v/V/^vvisual / line / block

Move

h j k lleft / down / up / right
w/b/eword fwd / back / end
0/^/$line start / 1st char / end
gg/Gfile top / bottom
{n}Ggo to line n
^d/^uhalf page down / up
%jump to matching bracket
f{c}/t{c}find / til char on line

Edit

xdelete char
dd/yydelete / yank line
dw/cwdelete / change word
p/Ppaste after / before
u/^rundo / redo
r{c}replace one char
Jjoin line below
>>/<<indent / outdent
.repeat last change

Search & Replace

/textsearch forward
?textsearch backward
n/Nnext / prev match
*/#search word under cursor
:%s/a/b/greplace a→b, whole file
:%s/a/b/gc…with confirm prompt

Visual Mode

y/d/cyank / delete / change sel
>/<indent / outdent sel
~ u Utoggle / lower / upper case

Windows & Buffers

:sp/:vspsplit horiz / vert
^w wcycle windows
^w hjklmove to window dir
^w qclose window
:bn/:bpnext / prev buffer
gt/gTnext / prev tab

Marks & Macros

m{a}/`{a}set / jump to mark a
"ayy/"apyank / paste register a
q{a}...qrecord macro into a
@a/@@run macro / repeat last

Files

:w/:q/:wqsave / quit / save+quit
:q!quit without saving
:e {file}open file
