herdr

Cheat sheet · Terminal multiplexer
prefix = Ctrl-b (default) — press it, release, then the key below. Capital letter = Shift.

Essentials

prefix cNew tab
prefix vSplit right (vertical)
prefix -Split down (horizontal)
prefix h j k lMove between panes
prefix wSwitch workspaces
prefix qDetach, keep running

Panes

prefix zZoom / unzoom pane
prefix xClose pane
prefix H J K LSwap pane in direction
prefix rResize mode
prefix [Copy mode

Tabs

prefix n/pNext / prev tab
prefix 1-9Jump to tab by number
prefix TRename tab
prefix XClose tab

Workspaces & Session

prefix NNew workspace
prefix WRename workspace
prefix DClose workspace
prefix gGoto picker
prefix bToggle sidebar

Copy Mode

prefix [Enter copy mode
h j k lMove cursor
w b eWord motions
{/}Paragraph up / down
^b/^fPage up / down (or PgUp/PgDn)
^u/^dHalf page up / down
v/SpaceStart selection
y/EnterCopy selection, exit
q/EscExit copy mode

With the default prefix, ^b enters prefix mode instead of paging up — remap the prefix if you page often. Mouse drag-select also works.

Prefix-Free Chords

ctrl+alt+…Safest modifier family for direct bindings
[keys]e.g. new_tab = ["prefix+c", "ctrl+alt+c"]

Avoid reserved chords: ctrl+alt+arrows (GNOME/Konsole), ctrl+alt+t (terminal launcher), ctrl+alt+l/a (KDE), ctrl+alt+f1–f12 (virtual consoles).
