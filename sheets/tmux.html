tmux

cheat sheet · shell command unless noted
prefix = Ctrl-b (default) — press it, release, then the key below.

Sessions

tmux new -s {name}new named session
tmux lslist sessions
tmux attach -t {name}attach to session
tmux kill-session -t {name}kill session
prefix ddetach from session
prefix slist & switch sessions
prefix $rename session

Windows

prefix cnew window
prefix ,rename window
prefix n/pnext / prev window
prefix 0-9select window by number
prefix wlist windows (interactive)
prefix &kill window

Panes

prefix %split vertical
prefix "split horizontal
prefix ocycle to next pane
prefix ←↑↓→move to pane in dir
prefix zzoom / unzoom pane
prefix xkill pane
prefix {/}swap pane left / right
prefix ^←↑↓→resize pane

Copy Mode

prefix [enter copy mode
↑↓/k jscroll up / down 1 line
^u/^dscroll up / down 1/2 page
^b/^fscroll up / down full page
g/Gtop / bottom of history
Spacestart selection
Entercopy selection, exit
prefix ]paste buffer
qquit copy mode

Misc

prefix :command prompt
prefix ?list all keybindings
prefix tshow clock
